# Disciplinas por Bloques — Grupo Nuwek / MEGA PLAST

Herramienta de clasificación y calendarización de actividades por puesto y empresa.

## Estructura

```
disciplinas-por-bloques/
├── index.html       # Estructura HTML
├── css/
│   └── styles.css   # Todos los estilos
├── js/
│   └── app.js       # Toda la lógica (Google Sheets, calendario, drag & drop)
└── README.md
```

## Uso

Abre `index.html` en cualquier navegador moderno.  
Los datos se sincronizan automáticamente con Google Sheets vía Apps Script.

## Tecnologías

- HTML / CSS / JavaScript vanilla (sin dependencias)
- Google Apps Script como backend
- Google Sheets como base de datos
